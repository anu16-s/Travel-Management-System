# Travel Management System

A web-based Travel Management System developed using Java, JSP, Servlets, and MySQL. The application allows users to browse and book national and international tour packages through an interactive and user-friendly interface.

## Features
- User Registration & Login
- National Tour Packages
- International Tour Packages
- Tour Booking
- Admin Management
- Responsive User Interface

## Technologies Used
- Java
- JSP & Servlets
- MySQL
- HTML
- CSS
- JavaScript
- Eclipse IDE
